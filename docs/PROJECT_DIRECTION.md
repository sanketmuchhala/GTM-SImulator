# Project Direction

## Product Name
GTM Simulation Lab

## One-Line Pitch
Simulate your GTM before launching outreach.

## What This Fork Is Becoming
A focused GTM simulation demo for founders and GTM teams.
Users input their product brief, ICP, and messaging strategy.
The system simulates buyer reactions, surfaces objections, and recommends
the best first outbound experiment — before a single real email is sent.

## What It Is Not
- Not a generic swarm simulation engine
- Not a social media analysis tool
- Not a marketing automation platform
- Not an email or outreach automation tool

## Target Users
- Early-stage B2B founders validating GTM before hiring SDRs
- GTM leads stress-testing messaging and ICP assumptions
- Sales engineers building GTM demos for prospects

## Primary Use Case
A founder enters their product brief, ICP, pricing, and target channel.
The system generates 12 buyer personas, tests 3 outreach message angles
against those personas, surfaces likely objections, and produces a
7-day outbound experiment recommendation — all before any real outreach.

## Demo Success Criteria
- Full flow completes in under 3 minutes
- Output is credible enough that a founder would reference it in a real GTM plan
- No auth, no database, no billing required to run the demo
- Fallback mock data exists so the demo works without API keys
